# 🗂️ Task Manager (Java GUI Application)

A **desktop-based Task Manager application** built using **Java**, designed to help users efficiently manage daily tasks with a simple and intuitive interface.

---

## 📌 Features

- ✅ Add new tasks  
- ✏️ Update/edit existing tasks  
- ❌ Delete tasks  
- 📋 View all tasks in an organized format  
- 🔐 User-friendly interface  
- ⚡ Fast and lightweight  

---

## 🛠️ Tech Stack

- **Language:** Java  
- **GUI:** Swing / AWT *(based on your implementation)*  
- **Architecture:** Modular (Core, IO, Utils)

## 🚀 Getting Started

### 1️⃣ Clone the repository
```bash
git clone https://github.com/<your-username>/TaskManager1.git
cd TaskManager1
```
2️⃣ Run the application
On macOS / Linux:
bash run.sh
Or manually:
javac -d bin src/taskmanager/**/*.java
java -cp bin taskmanager.Main
⚠️ Notes

You may see a warning:

uses or overrides a deprecated API

👉 This is not an error, the application will still run correctly.
